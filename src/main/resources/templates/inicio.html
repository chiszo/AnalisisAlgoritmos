Bienvenido al comparador de algoritmos de ordenación

Selecciona un método en el menú superior.

Ejemplos de estructuras de bucles y condicionales:

Caso 1

for(int i=0; i<n; i++) {          O(n)
    for(int j=0; j<n; j++) {      O(n)
        if(i == j)                O(1)
            matriz[i][j] = 1;     O(1)
        else
            matriz[i][j] = 0;     O(1)
    }
}

        Complejidad total: n×(n×O(1))=n×n = O(n²)

Caso 2

suma = 0;                        O(1)
for(int i=0; i<n; i++) {         O(n)
    if(i % 2 == 0)               O(1)
        suma += i;               O(1)
}
System.out.println(suma);        O(1)

        Complejidad total: O(1)+n×(O(1)+O(1))+O(1)
                            O(1)+O(n)+O(1) = O(n)

Caso 3

int i = n;                       O(1)
int contador = 0;                O(1)
while(i > 1) {                   O(log n)
    i = i / 2;                   O(1)
    contador++;                  O(1)
}
System.out.println(contador);    O(1)

        Complejidad total: O(1)+O(1)+O(logn×1)+O(1) = O(log n)
